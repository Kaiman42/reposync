# RepoSync 🚀

## Funcionalidades

- **Sem dependências**: Basta baixar o binário e rodar. Não precisa de pacotes extras.
- **Multiplataforma**: Funciona perfeitamente no Fedora 43 (KDE/Plasma) e Windows.

## Estrutura do Projeto

O projeto está organizado por sistema operacional:
- `windows/`: Contém o binário `.exe` e a lógica de UI do Windows.
- `linux/`: Contém o binário para Linux e a lógica de UI para Linux.
- Raiz: Código core em Go, configurações e documentação.
